# ApplyFlow 🚀

## Your Job Search, Simplified

ApplyFlow is a modern job application tracking platform that helps students and job seekers organize applications, schedule interviews, manage follow-ups, and track progress — all from one beautiful dashboard.

---

## 🌟 Features

### 📋 Application Tracking
Keep all your job applications organized in one place with real-time status updates.

### 📅 Interview Scheduler
Track interview dates, times, and follow-ups without missing important opportunities.

### 🔔 Smart Reminders
Get reminders for interviews, follow-ups, and application deadlines.

### 📊 Status Management
Update application status with visual indicators:

- ✅ Applied
- 🎯 Interview
- 🏆 Selected
- ❌ Rejected

### 🎨 Modern UI
Responsive and animated user interface built with Material UI and Framer Motion.
