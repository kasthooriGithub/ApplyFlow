ApplyFlow 🚀

Your Job Search, Simplified

ApplyFlow is a modern job application tracking platform that helps students and job seekers organize applications, schedule interviews, manage follow-ups, and track progress — all from one beautiful dashboard.

🌟 Features
📋 Application Tracking

Keep all your job applications organized in one place with real-time status updates.

📅 Interview Scheduler

Track interview dates, times, and follow-ups without missing important opportunities.

🔔 Smart Reminders

Get reminders for interviews, follow-ups, and application deadlines.

📊 Status Management

Update application status with visual indicators:

Applied
Interview
Selected
Rejected
🎨 Modern UI

Responsive and animated user interface built with Material UI and Framer Motion.

🛠️ Getting Started
Prerequisites

Make sure you have installed:

Node.js (v18 or higher)
npm or yarn
Firebase Project

⚙️ Installation
    1️⃣ Clone the Repository
    git clone <your-repository-url>
    2️⃣ Navigate to Project Folder
    cd ApplyFlow
    3️⃣ Install Dependencies
    npm install
    4️⃣ Start Development Server
    npm run dev

Application runs on: http://localhost:5173


🔥 Firebase Setup

Create a .env file in the root directory and add:

VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
📦 Build for Production
npm run build
